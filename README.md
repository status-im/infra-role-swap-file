# Description

Changes a [SWAP](https://en.wikipedia.org/wiki/Swap) file on Linux to avoid running out of memory.

# Configuration

```yaml
swap_file_path: '/main.swap'
swap_file_size_mb: 8192
swappiness: 10
```
